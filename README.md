# Hackovid
Web application to buy from different local shops in a single command

## Project's structure
Components:
* [Front](front). Web App.
* [Back](back). API.
* [Postgres](postgres). Initialization script for the Postgres DB.

## Build and run
### Locally
Execute: 
```
docker-compose up --build --remove-orphans
```

Then, go to http://localhost:7000.
